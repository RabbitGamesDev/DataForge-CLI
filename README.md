## 📖 Descripción
**DataForge CLI** es una herramienta de terminal diseñada para desarrolladores que buscan automatizar la auditoría de código, la documentación y la comprensión de arquitecturas complejas. Olvídate de perder tiempo analizando manualmente proyectos extensos; deja que la IA trabaje por ti.

---

## 🛠️ Comandos Principales

| Comando | Descripción |
| :--- | :--- |
| `scan` | Realiza un escaneo profundo de todo tu proyecto. |
| `explain` | Analiza y explica la lógica de un archivo específico. |
| `map` | Genera un mapa visual (ASCII) de tu arquitectura. |
| `ask` | Inicia una sesión de chat con el contexto de tu código. |
| `onboard` | Crea una guía de inicio para nuevos integrantes del equipo. |

💡 ¿Por qué usar DataForge?
⚡ Velocidad: Obtén auditorías y explicaciones en segundos.

🧠 Contexto inteligente: La IA no solo lee archivos, entiende cómo interactúan entre sí.

🛡️ Privacidad: Tu código es procesado localmente y tú tienes el control total de qué información se analiza.

🧩 Escalabilidad: Estructura modular diseñada para crecer junto a tus proyectos.
